Modern, high-performance portfolio website for Automora Tech - showcasing web design, AI automation, and digital solutions.

- **Single Page Application** - Smooth page transitions with custom routing
- **Responsive Design** - Optimized for all devices
- **Project Showcase** - Interactive portfolio cards with live project links
- **Team Section** - Meet the founders
- **Service Offerings** - Comprehensive service accordion
